海即数据 — 海不是背景，是四条读得出的数据通道

岛浮在海上，海面每一处读数都直接转录自账本与 substrate 分，没有一处是装饰。R7 起海面文法定为四通道，彼此正交：

暗度 = 深度
substrate · DC/均值轴 · col*=(1-uDepth)
色调 = 昼夜
全局 veil 0x0A1428 · α=t×0.66 · 载明度轴但全局作用(海无昼夜 uniform)
躁动 = 争议
被驳斥主张 · AC/纹理轴 · 零均值 chop 织纹
[RESERVED] flowline = 关系
未 ship · 卡 06 · 后续设计回合

频域正交是本卡的核心不变量：深度写均值(DC)——把整片水压暗；躁动写纹理(AC)——零均值高频织纹，眯眼看平均色调不变（AC1）。二者同格叠加互不串色（昼夜是独立的第三机制：全局 tone veil 0x0A1428·α=t×0.66，也写明度但作用于整场景，靠作用域而非频率与深度区分，非第四根正交轴——海 shader 无昼夜 uniform，scene-stage.ts buildToneOverlay/applyToneAndLights L811–814/887）。另一条老不变量仍在：深≠色，depth 独立于 domain-hue（sea.ts:100–106）。

Ch.1 深度暗化 sea depth = domain abstractness · DC / mean axis

substrate 抽象度 ∈ [0,1]（0 = 浅滩/偏应用，1 = 深水/偏形式）经 seaDepthAt() 变成一个只加暗、不加色的 alpha。海面着色器把它当乘法暗化因子；离岸越远沉得越暗，且下沉速度随 uDepth 增大（抽象岛更快跌进暗海）。下面两格同为平静海(0 被驳斥主张)，只有 substrate 不同：

// packages/renderer/src/sea.ts
export const SEA_DEPTH_MAX_ALPHA = 0.42;        // L88 · 最深水的最大暗化 alpha
export function seaDepthAt(substrate) {          // L107–111
  if (substrate == null || Number.isNaN(substrate))
    return { depth: null, overlayAlpha: 0 };  // 无分 → 不暗化(诚实缺席)
  const depth = clamp01(substrate);
  return { depth, overlayAlpha: depth * SEA_DEPTH_MAX_ALPHA };
}
// sea-mesh.ts · uDepth = seaDepthAt(substrate).overlayAlpha ∈ 0..~0.42
col = mix(col, uDeepColor, smoothstep(0.0,1.0,radial)*(0.28+0.55*uDepth)*(1.0-here)); // L129 离岸渐变，reach∝uDepth
col *= (1.0 - uDepth);                        // L154 · 就是这一乘：深＝暗，never a new hue
INVARIANT · 深≠色 depth 读的是真实 substrate 分，刻意不是 domain-hue 向量——独立的一根轴，永不是"旋转过的色相副本"（sea.ts:100–106）。substrate 为 null/NaN 时 overlayAlpha=0，宁可不画暗度也不编假深浅（honest absence beats decoration）。

岛身色相恒定 #C9D8E6（数理）；左浅滩 uDepth 0.076、右深水 uDepth 0.386 —— 同一色相、不同暗度，就是 depth ⊥ hue。

Ch.2 躁动 agitation contention = refuted-claim count · zero-mean chop weave · AC / texture axis

躁动 agitation = contention(被驳斥主张数)。争议不再压暗海面（那会跟深度抢明度轴），而是给水面加一层零均值网巾纹碎浪：两列逆向 ±45° 交叉 sin 织纹沿 uSeaColor↔uDeepColor 对比轴一加一减地重分布，不改平均明度——眯眼看色调不变（AC1），但纹理清晰起波。fbm zone 场把它聚成斑块、岸区 shoreExcl 硬排除（岸浪 foam 保持干净）、离岸 open 挡住陆地。被驳斥是 ever-refuted（账本无化解动词，一次 refute 即永久；是否加化解动词 advisor 已裁定属 scene 循环范围外、归未来 core 轮）。下面 0/1/2/3-被驳斥主张 四格条带与生产 shader 同参、深度恒定（substrate 0.45），变的只有织纹密度；1-refute(u=0.60)是金框主展品，u=1.0 仅作 3+ 端点。

// apps/web/src/scene/undertow.ts — 文件名保留；DATA 名仍是 contention（R7 只改了 VISUAL 名）
export function contentionFromRefuted(refuted) {   // L24–27 · 0→0, 1→0.60, 2→0.80, 3+→1.0
  if (refuted <= 0) return 0;
  return Math.min(1, 0.6 + (refuted - 1) * 0.2); }
export function refutedClaimCount(claims) {         // L37–40 · R7 Dim 1 单源：ghost==='refuted' 的 claim 数
  return claims.filter(c => c.ghost === 'refuted').length; }
// GeneratedIslandScreen.tsx · contention: contentionFromRefuted(refutedClaimCount(projected))  (L216–217)
//                            <PixiScene agitation={seaStats?.contention ?? 0} />                (L364)
// scene-stage.ts L1055 · setAgitation(on: boolean|number) → uAgitation = v   (R7 Dim 2, was setUndertow)

// packages/renderer/src/pixi/sea-mesh.ts (L168–175) · 争议 = 表面 CHOP（零均值），不是压暗
if (uAgitation > 0.001) {                                   // 零保持门(AC5)
  float open      = 1.0 - here;
  float shoreExcl = 1.0 - smoothstep(0.05, 0.30, coast);        // Gen-D · 岸带硬排除(AC6)
  float zone      = smoothstep(0.46 - 0.08*uAgitation, 0.74, fbm(w*0.006 - vec2(uTime*0.045, uTime*0.03)));  // R7 tip: 更大斑块
  float chopA     = sin((w.x + w.y) * 0.095 - uTime * 1.15);        // +45° 列
  float chopB     = sin((w.x - w.y) * 0.078 + uTime * 0.95);        // −45° 逆向列
  float chop      = chopA * chopB * (0.68 + 0.32 * noise(vec2((w.x+w.y)*0.05, (w.x-w.y)*0.0055)));  // 各向异性噪声 ~9:1 沿织纹拉伸:调幅不散方向
  col += (uSeaColor - uDeepColor) * chop * (3.5*uAgitation) * zone * shoreExcl * open; // 沿对比轴重分布，不写明度均值
}
INVARIANT · 零均值 / 频域正交 chop 是两列 ±45° sin 的积 × 各向异性噪声，sRGB 帧缓冲值零均值（sRGB gamma 凸性带来的线性亮度漂移 ≤0.6%，sub-JND）；沿 (uSeaColor − uDeepColor) 对比轴一加一减，眯眼看平均色调不变（AC1）。躁动是 AC（纹理/高频），深度是 DC（均值/暗度）——频域正交，可同格叠加互不串。零保持(AC5)：uAgitation>0.001 门 + (3.5*uAgitation) 因子 → u=0 与无躁动逐字节相同。岸区(AC6)：shoreExcl 硬排除岸带。
HISTORY · R6→R7 R6 此通道曾是压暗漩涡（col = mix(col, col*0.5, swirl)，写明度轴）。R7 因 H3「明度轴超载」（深度也在明度轴，争议一压暗就与深度串味、眯眼分不清）裁定改为零均值躁动，把争议挪到纹理轴与深度频域正交。传递曲线 contentionFromRefuted 未变（R6 Lever 2 标定值沿用）。

深度全格恒定 substrate 0.45（uDepth 0.189）；uAgitation 随被驳斥主张 0 → 0.60 → 0.80 → 1.0，织纹密度(coverage)随之升。每格「眯眼 squint →」色块是该格开阔海平均色调：四块几乎同色 = 明度中性(AC1)。金框 = 1-refute 主展品；最右 3-refute = 满量程端点。reduced-motion 下取静帧——织纹是空间签名，冻结仍读得出躁动(AC8)。
