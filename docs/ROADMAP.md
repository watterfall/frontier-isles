# Frontier Isles · Roadmap Assessment

_As of 2026-07-18, after Phase B closed (§8 accepted), the richness + world-map campaigns, vertical/nested-world passes, continuous-world exploration, island-depth upgrade, the cross-domain research-comparison reset, and the first executable-model layer. V1 nested world **landed** (`548fa26`); the frontier atlas grew 26→78→128 islands and received the comfortable-at-every-tier and world-tier legibility passes. L0.5 provides real continuous flight, question-led courses, altitude alignment, current sampling, survey notes, docking, and pose-preserving return; its versioned local-first notebook preserves and exports those traces. Inside generated L1 islands, five evidence-gated districts form a progressive local map, while every visible station projects distinct navigable floors from real questions, literature, evidence, artifacts, residents, ledger statistics, and active mechanisms. The former taxonomy-first Structure Lens is no longer the primary relation experience: one default research-comparison view fuses human mechanism mappings, curated mathematical forms, and signed ledger currents without creating edges from theme or proximity. It opens with actual shared statements and study pairs, supports concrete-problem search, and narrows only then into a two-problem human validation workbench. Relation types remain in the graph and renderer but move behind progressive disclosure; the primary language is `从别的研究里找办法 / 能借用什么 / 哪里不能照搬 / 怎么验证`. Signed ledger paths open a source-preserving dossier: the original claim/material, its evidence, each response body, response evidence, actor/time, discriminating test, and honest historical gaps remain separately inspectable. A desktop human can support or challenge an anchored claim from another island through one transactional ledger write. Separately, a learner can now define, predict, run, transfer, and limit a deterministic model instead of receiving an answer: two deterministic rule families are each reused across four interpretations, with the shared update and the point where the analogy fails made explicit. Notebook v4 stores learner-owned runs locally; neither a simulation result nor its interpretation transfer writes a research edge. Mobile keeps the complete research read truth and now permits only this local, non-ledger model notebook write. Primary island vocabulary is functional (`连接工作台`, `开放讨论`, `问题对照`, `证据`, `方法试验`). Companion to `architecture.md` v3.0 §8, `depth-plan-v1.md`, `depth-plan-v2.md`, `atlas-world-plan.md`, `nested-vertical-world-v1-2026-07-11.md`, `island-depth-plan-2026-07-18.md`, `connection-field-v1-2026-07-18.md`, `connection-dossier-v1-2026-07-18.md`, `connection-language-reset-v2-2026-07-18.md`, and `executable-model-layer-v1-2026-07-18.md`; this file records **actual** status and the prioritized path forward._
