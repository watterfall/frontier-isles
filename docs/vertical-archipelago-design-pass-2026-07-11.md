# 垂直群岛设计迭代 · 2026-07-11

## 方向

采用“立体界画悬空群岛”：保留原有海图坐标、语义缩放、暖纸矿物色和列表孪生体，以正交伪 3D 增加纵向空间，而不引入自由旋转的真 3D 相机。

## 高度契约

- 高度是 **place-plane 制图位置**，不是成熟度、质量、活跃度或排名。
- 将现有地图的南北顺序稳定折叠为高空、中空、低空三层；slug 只在坐标相同时作为确定性排序键。
- Stage、成员、事件、休眠、离群与 resolved 仍使用原有建筑、居民、烟迹、苔藓、发光和灯塔通道表达。
- 空中航线只来自现有跨领域 current；云层只从既有 fog cell 生成。

## 视觉与交互

- 岛体：原有软丘海岸线保持不变，新增矿物岩层式岛底、岩脉、空气阴影与风纹。
- 世界：跨领域洋流抬升为空中航线，节点形成风结；雾场分成前后两层云带，制造遮挡和空气透视。
- 相机：继续使用一台连续相机完成世界 → 区域 → 岛屿 → 岛内下钻；靠岸目标使用投影后的真实屏幕坐标。
- 筛选：桌面和移动端均可查看全空域、低空、中空或高空；筛选是明确显隐，不用低透明度制造 GPU 合成歧义。
- 标签：近景副标题增加空层名称；高度图例明确声明“不是研究排名”。

## 响应式与降级

- 桌面由 Pixi 绘制完整 2.5D 图集，保留 SVG 无 GPU 回退。
- 移动端使用独立 SVG 正交微缩图，保留三层筛选、键盘语义和列表孪生体，不加载桌面 Pixi 地图。
- reduced-motion 下镜头仍给出快速落点反馈，但取消非必要的循环和位移动画。

## 参考原则

垂直维度必须改变“看见哪里、如何抵达、何时揭示信息”，不能只把岛屿向上移动。实现因此同时覆盖高度分层、空中关系、遮挡、连续靠岸和空层导航，而不是单独增加投影阴影。

## V1 · 纵向嵌套世界

在本轮基础上继续加入连续高度、群岛内部的主岛—伴岛层级、真实岛间空路与渐进显现。主岛由群岛几何中心确定，只是导航入口，不是科研排名；伴岛在中景保持潜伏、近景才可见并参与命中。完整契约与验收见 `nested-vertical-world-v1-2026-07-11.md`。
