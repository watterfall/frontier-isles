# 世界探索设计全面提升 · 2026-07-11

## 目标

在不破坏 ledger 投影、日夜语义、列表孪生体和无 GPU 兜底的前提下，使真实产品至少恢复旧版原型的空间层次，并向开放世界的探索引导靠近。

## 从旧版恢复的不可退化资产

- L0：远景地貌、知识气候、区域、岛屿聚落、活动痕迹五层同时成立。
- L1：九个 civic station 仍是聚落骨架；路径、居民、植被、岸线、Claim 建筑构成生活层。
- Claim：一条主张是一栋可成长的小建筑；地基、楼层、屋顶和幽灵脚手架分别转录公开、复现、共识和驳斥/退回。

## 开放世界式探索规则

1. 远处先给地标和活动线索，不用任务图标替代空间。
2. 世界层 → 区域层 → 岛屿层只用一台连续相机；搜索结果也先飞近再靠岸。
3. 道路必须解释空间：岛内欲望径连接真实可见站点，不能画无数据含义的关系网。
4. 大中小地标共存：知识气候 / 区域，岛屿聚落 / Landmark，Claim / 居民 / 环境物。
5. 密度来自 stage、events、members、domain；装饰不能伪装成进度或排名。

## 本轮落地

- L0 岛屿增加数据驱动的建筑、植被、居民刻痕、活动烟迹、水纹和领域远景等高线。
- 增加真实的领域聚焦、缩放、全图复位、层级读数和搜索飞行。
- L1 补通生成器居民到 Pixi 的数据链，补齐树、竹、莲池、灯笼、礁石等独立轮廓。
- L1 依据可见站点生成欲望径；Claim 从碑体恢复为细塔建筑，同时保留协议语义。
- Pixi 图集增加运行时错误边界；失败时回落 SVG 海图，不把整页留白。

## 垂直群岛增量

- L0 从平面海图升级为正交伪 3D：原有南北 place-plane 稳定折叠成低空、中空、高空，不把高度解释为排名。
- 岛屿增加岩层岛底、空气阴影和风纹；真实跨领域 currents 抬升为空中航线，fog cells 转写为前后云层。
- 桌面和移动端均提供空层筛选；近景标签标注空层，图例明确“高度是制图位置，不是研究排名”。
- 连续相机、搜索靠岸、领域聚焦、SVG 兜底与列表孪生体保持不变。
